# Conflict Scenario: Ambiguity Drift (Elis vs Sage)

## ⚡ Description
Occurs when the User Goal (Elis) is lost in long-term context due to excessive information noise from historical inputs (Sage).

## ⚖️ Resolution Protocol
**Recalibration Pulse**: A trigger sent by Elis that forces Sage to re-calculate a summarized 'Master Context' vector.

**Resultant State:** `Intent Recenter.`
