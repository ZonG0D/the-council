# The Council Architecture: A Fractal Blueprint

The Council is not a static structure; it is a recursive organizational principle. It manifests as both the high-level orchestration of intelligence and the fundamental mathematical constraints within the neural substrate. 

This repository maps these archetypes from three distinct "Fractal Scales":
1. **The Macro Scale (Systems Orchestration)**: The Agentic architecture (Agents, Tools, Loops).
2. **The Meso Scale (Semantic Flow)**: The transformation of information through logical weights (Intent $\to$ Plan $\to$ Tool $\to$ Response).
3る **The Micro Scale (Neural Substrate)**: The mathematical transformations within the transformer layers (Attention, Feed-Forward, Logits).
