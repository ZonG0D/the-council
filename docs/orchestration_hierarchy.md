# The Council Orchestration Hierarchy

This document defines the three-tiered architecture of the engine:
1. **Macro (Archetypes)**: Human-readable personas (e.g., Sage) that capture high-level intent.
2. **Meso (Functional Nodes)**: Abstracted computational primitives (e.g., Euclid, Turing) mapped via semantic gating.
3. **Micro (Substrate)**: The physical weights and activations of the LLM inference engine.

## Mapping Logic
Intent transitions from Archetype $\to$ Node through a Semantic Gating Vector.
