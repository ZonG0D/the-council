# The Council: Homeostatic Control Loops (Cybernetic Architecture)

This document formalizes the application of **Negative Feedback Loops** across the three fractal scales of The Council orchestration framework. It defines how stability, autonomy, and course correction are maintained from high-level intent down to mathematical convergence.

---

## 🔄 Overview: The Cybernetic Principle

The Council operates as a self-regulating, multi-layered control system. Every level (Macro, Meso, Micro) utilizes the fundamental cybernetic loop—**Stimulus $\to$ Sensor $\to$ Control $\to$ Effector $\to$ Response**—to maintain operational homeostasis and achieve high-fidelity goal execution.

---

## 🏛️ I. Macro Scale: Orchestration & Agency
*The "Who" – Managing high-level system objectives (Goal Alignment).*

| Component | Mapping to Council Architecture | Description |
| :--- | :--- | :--- |
| **Stimulus** | Goal Discrepancy ($\Delta G$) | Detection of a gap between current state $S_{current}$ and targeted goal $G$. |
| **Sensor** | **Argus (Universal Observer)** | Continuous monitoring via process observation, tool-call status, and environment telemetry. |
| **Control Center** | **Elis / Sage** | High-level strategic reasoning; evaluation of task trajectory against the roadmap. |
| **Effector** | **Silas / Weaver** | Re-routing tasks, spawning subagents (delegation), or modifying orchestration protocols. |
| **Response** | Goal Realization | Restoration of systemic alignment with the mission objective. |

### ⚠️ Primary Failure Mode: "Strategic Drift"
*When the Macro loop fails to detect divergent agent behaviors, leading to task completion that deviates from original intent.*

## 🌊 II. Meso Scale: Semantic Flow & Cognitive State
*The "How" – Managing information transformation (Semantic Coherence).*

| Component | Mapping to Council Architecture | Description |
| :--- | :--- | :--- |
| **Stimulus** | Semantic Drift ($\text{dist}(V_t, V_{target})$) | A shift in latent representation or conversation context away from the intended topical manifold. |
| **Sensor** | **Lyria / Ariadne (Context Monitor)** | Real-time analysis of attention weights and embedding distances across deep token history. |
| **Control Center** | **Semantic Gating Logic** | Decision-making layer evaluating if current discourse path is logically consistent or hallucinating. |
| **Effector** | Contextual Injection / Prompt Tuning | Manipulating input tokens, pruning irrelevant context, or adjusting prompt constraints (e.g., Aegis/Prism). |
| **Response** | Coherence Restoration | Return of the semantic manifold to its targeted trajectory within a defined $\epsilon$ threshold. |

### ⚠️ Primary Failure Mode: "Contextual Hallucination"
*When Meso error correction is too weak, allowing local token-wise coherence to mask global logic loss.*

## 🧮 III. Micro Scale: Neural Substrate & Mathematical Logic
*The "What" – Managing parameter optimization (Numerical Convergence).*

| Component | Mapping to Council Architecture | Description |
| :--- | :--- | :--- |
| **Stimulus** | Gradient Error / Loss ($\mathcal{L}$) | The mathematical difference between predicted and target probability distributions. |
| **Sensor** | **The Optimizer (SGD/Adam)** | Real-time calculation of loss gradients within the backpropagation loop. |
| **Control Center** | Calculus-Based Logic | Optimization algorithms functioning as the central regulator of parameter space. |
| **Effector** | Weight Update / Backpropagation | The physical adjustment of weights $(\theta)$ to minimize error in subsequent forward passes. |
| **Response** | Convergence | Minimization of loss; optimization of predictive accuracy and distribution stability. |

### ⚠️ Primary Failure Mode: "Gradient Vanishing/Explosion"
*When mathematical updates become too small or too large, breaking the loop's ability to restore balance.*
